# Yu
一个基于 Tauri 与 Vue 3 开发的轻量桌面小工具。✨ 主要功能  目前它还处于比较初级的阶段，仅包含了几个我最常用的功能：      任务清单：随手记录待办，保持思路清晰。      进度追踪：通过进度条直观感受任务完成度。      番茄时钟：简单的计时工具，辅助进入专注状态。  🛠️ 技术实现  为了追求极致的轻量化和较低的资源占用，我选择了以下技术栈：      框架：Tauri (Rust 后端)。      前端：Vue 3 + Vite。      风格：追求扁平化的极简线条感，支持背景透明与毛玻璃效果。  ⚙️ 说明  这只是一个纯粹出于个人兴趣和学习目的开发的练习作品。代码可能还不够优雅，功能也在慢慢完善中，欢迎交流和指点。
